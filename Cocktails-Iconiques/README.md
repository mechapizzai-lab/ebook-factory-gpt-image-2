# Cocktails Iconiques

Livre visuel de recettes: 20 cocktails classiques, beaux et faisables a la maison.

## Format

- Type: livre de recettes visuel
- Langue: francais
- Pages prevues: 22 images pleine page
- Dimensions cible: 1600x2560 px
- Cible: KDP ebook fixed-layout

## Direction artistique

Style constant sur toutes les recettes:

- photographie editoriale premium
- cocktails tres lumineux et appetissants
- fond de bar moderne, propre, legerement floute
- typographie elegante mais tres lisible
- blocs simples: ingredients, methode, astuce facile
- mini-panneaux visuels pas-a-pas pour montrer exactement les gestes
- ambiance accessible, pas laboratoire de mixologie

## Structure

1. Couverture
2. Sommaire
3. 20 pages recettes

## Workflow

1. Valider le prototype dans `prototype/`.
2. Generer les 20 pages recettes depuis `prompts/`.
3. Normaliser les images en `1600x2560`.
4. Assembler l'EPUB fixe et les sorties KDP dans `output/`.

## Note

Recettes reservees aux adultes. A consommer avec moderation.
