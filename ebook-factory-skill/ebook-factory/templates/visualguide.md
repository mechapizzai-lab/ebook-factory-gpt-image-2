# Visual Guide / How-To Template

## Page types

### Step-by-step page
```
Layout:
┌─────────────────────────┐
│  SECTION TITLE          │
├──────┬──────┬───────────┤
│ IMG1 │ IMG2 │  IMG3     │
│ Step │ Step │  Step     │
│  1   │  2   │   3       │
├──────┴──────┴───────────┤
│  Detailed explanation    │
│  and tips for this       │
│  technique...            │
└─────────────────────────┘

Prompt pattern:
"[style anchor] Step-by-step instructional illustration:
[specific action being demonstrated],
clear and educational, numbered step visual,
clean background, good contrast for readability,
[visual style], professional how-to guide quality"
```

### Before/After comparison page
```
Layout:
┌───────────┬─────────────┐
│  BEFORE   │   AFTER     │
│  [img]    │   [img]     │
├───────────┴─────────────┤
│  What changed and why    │
└─────────────────────────┘
```

### Infographic page
```
Prompt pattern:
"[style anchor] Educational infographic about [TOPIC],
clean data visualization, icons and labels,
organized layout with clear hierarchy,
[visual style], professional guide illustration"
```

## Style anchors

### Clean instructional
```
"Clean professional instructional illustration style,
precise lines, neutral background, clear labeling areas,
consistent icon style, educational diagram quality,
blue and gray color scheme with orange accents for highlights"
```

### Lifestyle photography style
```
"Lifestyle photography style, natural lighting,
warm color grading, styled scenes with props,
overhead and 45-degree angles, editorial quality,
clean modern aesthetic with plenty of white space"
```

# Art Book / Portfolio Template

## Page types

### Full-bleed art page
```
Layout: Image fills entire page, no margins

Prompt pattern:
"[style anchor] [artwork description],
[composition and mood], [technique/medium],
high detail, professional art quality,
suitable for full-page art book reproduction"
```

### Gallery page (2-4 images)
```
Layout:
┌──────┬──────┐
│      │      │
│ IMG1 │ IMG2 │
│      │      │
├──────┼──────┤
│      │      │
│ IMG3 │ IMG4 │
│      │      │
└──────┴──────┘
Optional: thin captions below each image
```

### Artist statement / Text page
```
Layout: Minimal text on clean background
- Pull quote in large type
- Short paragraph of context
- White space dominant
```

## Style anchors

### Fine art collection
```
"Fine art quality, museum-grade reproduction,
rich color depth, precise detail rendering,
gallery-worthy composition, sophisticated palette,
professional art book standard"
```

### Street art / Urban
```
"Urban street art aesthetic, bold graphic style,
graffiti influence, vibrant saturated colors,
raw textured surfaces, contemporary art book quality,
expressive and energetic compositions"
```
