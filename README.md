## 🎯 C'est quoi cette "factory" ?

`ebook-factory-gpt-image-2` est un **pipeline complet** qui transforme une idée en ebook illustré et prêt à publier. C'est un projet pensé pour ceux qui veulent **monétiser l'IA** via la création de produits digitaux.

### ⚡ Ce que la factory fait pour toi

- ✍️ **Génère le contenu** texte structuré
- 🎨 **Crée les illustrations** avec GPT Image 2
- 📖 **Assemble le tout** dans un ebook propre
- 🚀 **Sort un produit** prêt à vendre

> 💰 **Cas d'usage rentable** : créer une collection d'ebooks illustrés (enfants, niches spécifiques, formations) à publier sur Amazon KDP, Etsy, ou ton propre site.
